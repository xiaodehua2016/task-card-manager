## 🚀 发布流程

### 1. 准备发布
```bash
# 检查当前状态
git status

# 提交所有更改
git add .
git commit -m "准备发布 v3.0"

# 运行发布脚本
./deploy/release-v3.0.sh  # Linux/macOS
# 或
deploy\release-v3.0.bat   # Windows
```

## 🔄 回退计划

如果发现问题需要回退：

### 快速回退
```bash
# 回退到上一个版本
git revert HEAD
git push origin main
```

### 完全回退
```bash
# 回退到v2.0版本
git reset --hard v2.0.0
git push --force-with-lease origin main
```
